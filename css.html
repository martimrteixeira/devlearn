Introdução ao CSS

Progresso: 0%

CSS (Cascading Style Sheets) é a linguagem usada para definir o visual das páginas web. Com CSS controlas cores, tamanhos, espaçamentos e layouts.

HTML cria a estrutura, CSS cria o estilo.

Sintaxe básica do CSS

Um estilo CSS é composto por seletor, propriedade e valor:

seletor {
  propriedade: valor;
}

Cores

As cores podem ser definidas por nome, hexadecimal ou RGB:

color: red;
color: #2563eb;
color: rgb(37, 99, 235);

Fontes

Exemplo de propriedades para texto:

font-family: Arial;
font-size: 18px;
font-weight: bold;

Box Model

Todo elemento HTML é uma caixa composta por:

  • Content
  • Padding
  • Border
  • Margin

Layout

CSS permite criar layouts com:

  • display: block / inline
  • flexbox
  • grid
display: flex;
justify-content: center;
align-items: center;
